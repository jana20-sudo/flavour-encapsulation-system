# 🧪 FlavorEncap AI

### Advanced Flavor Encapsulation Simulation & Optimization Platform

FlavorEncap AI is an intelligent, simulation-driven web application designed for modeling and optimizing spray drying processes in flavor encapsulation.

It combines **process engineering**, **AI reasoning**, to help users analyze, optimize, and understand flavor encapsulation systems in real time.

This project was **developed with the assistance of AI tools** , combined by domain knowledge in food engineering and process modelling.

---

## 🚀 Key Features

### 🔬 1. Spray Dryer Simulation Engine

* Physics-based modeling of:

  * Encapsulation Efficiency (EE)
  * Moisture Content
  * Particle Size (D50)
  * Water Activity (Aw)
  * Glass Transition Temperature (Tg)
* Based on industry-relevant equations (Arrhenius, psychrometrics, etc.)

---

### 🧠 2. Intelligent Warning & Diagnosis System

* Detects process issues automatically
* Explains **why the issue occurs**
* Suggests **specific parameter changes**
* One-click **Apply Fix → Auto Optimization**

---

### ⚙️ 3. Auto Optimization Engine

* Searches optimal process conditions
* Balances:

  * Efficiency ↑
  * Moisture ↓
  * Flavor loss ↓
* Converts simulation into a **decision-making tool**

---

### 🤖 4. AI Assistant (Hybrid Intelligence)

* Rule-based engineering logic + AI reasoning
* Provides:

  * Process explanations
  * Improvement suggestions
  * Optimization guidance

---

### 📊 5. Interactive Visualization

* Real-time updating charts
* Multi-parameter analysis
* Performance comparison across runs

---

### 📈 6. Sensitivity Analysis

* Shows how each parameter affects:

  * Efficiency
  * Moisture
  * Stability
* Helps in **process understanding and control**

### 💾 7. Experiment Tracking

* Save and compare runs
* Analyze trends over time
* Supports data-driven decision making

---

## 🏗️ System Architecture

Frontend:

Backend:

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/jana20-sudo/flavour-encapsulation-system
```

### 2. Install dependencies

```bash
pip install flask flask-cors numpy scipy scikit-learn
```

### 3. Run the backend

```bash
python simulator.py
```

### 4. Open in browser

```
http://localhost:5000
```
